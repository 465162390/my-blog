# my-blog

#### 使用Vue全家桶、element-ui、PHP、Mysql开发的我的博客。 包括前、后台页面

#### 前台页面包括关于、文章页、分类、标签、留言板等
1、 用户可以对喜欢的文章进行点赞、评分或者二维码分享。
2、 按标签和分类对文章进行了筛选

#### 后台登录
1、 博主用户可以观看到博客每天的访问量，新增点赞评论等数据
2、 在后台可以对博客用户文章等进行管理

##### 博客对移动端也进行了响应式的适配
##### 博客集成了 微博、qq、github等第三方登录，有访客统计。
